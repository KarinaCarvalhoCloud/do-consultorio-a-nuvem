2 Cursos Essenciais e Gratuitos para Iniciar em Tecnologia

Publicado em 20 de outubro de 2025 • 5 min de leitura

Se você está vindo de uma área completamente diferente, como eu, saber por onde começar a estudar pode ser esmagador. Estes dois cursos foram fundamentais para me dar a base necessária em lógica de programação e entender o vocabulário da TI antes de mergulhar em Cloud Computing.

1. Fundamentos de Tecnologia

Primeiros Passos em Tecnologia

EV.G - Escola Virtual Fundação Bradesco (Gratuito)

Este curso é perfeito para quem está no zero. Ele cobre a história, as áreas de atuação, hardware, software, e noções de rede e segurança. É o ponto de partida ideal para alinhar sua mente com a lógica da tecnologia.

2. Lógica e Automação

Introduction to Python Programming

Santander Open Academy (Gratuito)

Aprender Python é a "chave" para automação em Cloud. Este curso te dá a introdução necessária à sintaxe e aos conceitos de programação (variáveis, loops, funções). Ter essa base faz toda a diferença nos estudos de Cloud e DevOps.

Acessar Curso →

Próximos Passos (Recomendações)

Após dominar esses fundamentos, eu sugiro focar em:

  • **Certificação AWS Cloud Practitioner:** Para entender os serviços da Nuvem.
  • **GIT e GitHub:** Essencial para controle de versão e portfólio.
  • **Linux Essentials:** Para ter fluência no sistema operacional mais usado em servidores Cloud.
